Proyecto para el curso de Inferencia Causal. Se presentan una carpeta con aplicaciones de regresión lineal multivariado, el modelo logit y el modelo probit, así como la base de datos desarrollada a partir de la ENAHO 2024. El tema general se basa en evaluar el efecto de hablar una lengua indígena con el nivel de confianza en la policía.
